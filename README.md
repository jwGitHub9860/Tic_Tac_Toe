# Project: Tic_Tac_Toe

### Description:

The tic-tac-toe project allows the user to play a game of tic-tac-toe.

### Instructions:

1. Choose to play either a **_1_** or **_2_** player game
2. Choose which game piece to use: ```X```'s or ```O```'s
3. **PLAY THE GAME**
